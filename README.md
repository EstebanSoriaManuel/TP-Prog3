**📄 Descripción**
Sistema de consola para procesar distintos métodos de pago de clientes. Registra pagos con información del titular y monto, aplica comisiones o cargos adicionales según el medio utilizado, y muestra el monto final de cada operación.

**✅ Funcionalidades**
*Registrar pagos con distintos métodos (tarjeta, transferencia, efectivo, etc.)
*Mostrar información del titular y monto original
*Procesar la operación y calcular cargos adicionales por tipo de pago
*Mostrar el monto final luego de comisiones
*Confirmación del resultado de la operación
